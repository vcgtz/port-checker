# Port Checker

A simple CLI tool to check if a port is available on localhost.

## Usage

```bash
npx @vcgtz/port-checker check <port>
```

## Examples

```bash
npx @vcgtz/port-checker check 5173
# Port 5173 is not available

npx @vcgtz/port-checker check 8080
# Port 8080 is available
```
